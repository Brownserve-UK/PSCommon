# Brownserve.PSCommon

[![release](https://github.com/Brownserve-UK/PSCommon/actions/workflows/release.yaml/badge.svg?branch=main)](https://github.com/Brownserve-UK/PSCommon/actions/workflows/release.yaml)

The foundational PowerShell module for Brownserve projects. It provides a set of core utilities shared across all other Brownserve PowerShell modules and project repositories, covering common operations such as content manipulation, file and path management, configuration loading, OS detection, YAML conversion, Markdown handling, and more.

The module is published as both a NuGet package and a PSGallery module for easy consumption in CI/CD pipelines and local development environments.

## What's included?

Please see the [module documentation](https://docs.brownserve.co.uk/Brownserve.PSCommon/) for a full list of cmdlets and their usage.

## How to use

See the [getting started guide](https://docs.brownserve.co.uk/Brownserve.PSCommon/getting-started/) for installation and setup instructions, or the [module reference](https://docs.brownserve.co.uk/Brownserve.PSCommon/reference/Brownserve.PSCommon/) for a full list of available cmdlets.
